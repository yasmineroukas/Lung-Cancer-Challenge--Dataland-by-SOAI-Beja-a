# 🏥 Patient Survival Prediction AI

## 📌 Overview

This project focuses on developing an AI model capable of determining a patient's survival chances based on comprehensive medical and personal data. By leveraging data science and real-world medical insights, this model aims to provide accurate and meaningful predictions to assist in healthcare decision-making.

## 🚀 Features

### Data-Driven Predictions:
Utilizes medical and personal data to estimate survival probabilities.

### Advanced Machine Learning: 
Implements cutting-edge algorithms for improved accuracy.

### Explainable AI: 
Provides insights into factors influencing survival predictions.

Scalable & Adaptable: Can be refined with new datasets for better performance.

## 📊 Dataset

The model is trained on:

Patient medical records (e.g., lab results, diagnoses, treatments)

Demographic and lifestyle factors

Historical survival data

## 🏗️ Installation

To set up the project, follow these steps:

### Clone the repository:

git clone https://github.com/yasmineroukas/Lung-Cancer-Challenge--Dataland-by-SOAI-Beja-a.git
cd patient-survival-ai

### Install dependencies:

pip install -r requirements.txt

### Run the model:

python main.py

## ⚙️ Usage

Prepare the dataset and place it in the data/ directory.

### Train the model:

python train.py

### Make predictions:

python predict.py --input patient_data.csv
